## **ОБЛАСТЬ ТЕХНИКИ**

Настоящее изобретение относится к приводам мягких роботов, а конкретнее — к **электромагнитным искусственным мышцам**, содержащим низковольтные электромагнитные катушки, встроенные в эластичную композитную матрицу. Система обеспечивает **аналоговое управление жесткостью, бесшумную активацию и биомиметическое движение** для использования в гуманоидных роботах (андроидах), протезах конечностей, аниматронных существах и тактильных интерфейсах — без применения моторов, пневматики или высоковольтных диэлектриков.

---

## **УРОВЕНЬ ТЕХНИКИ**

### 1. Ограничения известных решений

Традиционные роботизированные приводы основаны на:

- **Электродвигателях + сухожилиях**: Жесткие, шумные, низкая комплаентность.  
- **Пневматических/гидравлическиых системах**: Требуют компрессоров, склонны к утечкам, медленный отклик.  
- **Сплавах с памятью формы**: Низкий КПД, медленное охлаждение, ограниченный срок службы.  
- **Диэлектрических эластомерных приводах (DEA)**: Требуют 1–10 кВ, риск электрического пробоя, хрупкие электроды.

Ни одно из них не воспроизводит **переменный импеданс**, **энергоэффективность** и **бесшумную работу** биологической мышцы.

### 2. Биологическая основа

Человеческая мышца модулирует тонус посредством нервных сигналов — мягкая в покое, упругая при сокращении. Мимические движения лица используют антагонистические пары мышц (например, скуловая и угловая мышца рта) для тонкого контроля.

По-прежнему существует потребность в **твердотельной, безопасной, низковольтной искусственной мышце**, имитирующей такое поведение.

## **КРАТКОЕ ИЗЛОЖЕНИЕ СУЩНОСТИ ИЗОБРЕТЕНИЯ**

Изобретение представляет собой **электромагнитную искусственную мышцу (EM Muscle)**, включающую:

- Две или более **плоских спиральных электромагнитных катушек**  
- Полностью встроенных в **мягкую силиконовую матрицу** (например, Ecoflex 00-30)  
- Питаемых от **постоянного тока или ШИМ 0–12 В**  
- Опционально усиленных **магнитными частицами** (железо/феррит)

При подаче тока магнитные силы между катушками создают **внутреннее предварительное напряжение**, увеличивая эффективную жесткость материала-носителя или вызывая макроскопическую деформацию.

**Направление магнитного взаимодействия — притяжение или отталкивание — определяется полярностью подаваемого тока.** Это позволяет реализовывать как **сжимающие**, так и **расширяющие** режимы работы одной и той же конструкции, расширяя диапазон возможных движений и конфигураций.

**В дополнительном варианте исполнения между или внутри катушек размещается подвижный ферромагнитный сердечник (например, цилиндрический поршень из железа, пермаллоя или композита на основе магнитного порошка), заключённый в эластичную оболочку.** Под действием магнитного поля сердечник перемещается линейно, обеспечивая направленное выдвижение, толчок или тактильную обратную связь — при сохранении общей мягкости конструкции.

Несколько EM Muscles могут быть организованы в **антагонистические конфигурации** для обеспечения естественных движений в лицевой зоне андроидов, конечностях или биомиметических существах.

**Ключевые преимущества:**

- **Без высокого напряжения** (безопасно для взаимодействия с человеком)  
- **Бесшумная, твердотельная работа**  
- **Аналоговое управление жесткостью** (плавный переход от мягкого к твердому)  
- **Масштабируемость** от мм (пальцы) до см (руки, лицо)  
- **Двунаправленная активация** (притяжение/отталкивание без механической переконфигурации)  
- **Опциональное линейное действие** через подвижный магнитный поршень

### 2. Принцип действия

Магнитная сила между катушками выражается формулой:
\[
F \approx \frac{\mu_0 N^2 I^2 A}{2 d^2}
\]
Где:
- \( \mu_0 \) = магнитная постоянная  
- \( N \) = количество витков  
- \( I \) = ток (типично 0,1–0,5 А)  
- \( A \) = эффективная площадь  
- \( d \) = расстояние между катушками  

**Знак силы (притяжение или отталкивание) зависит от относительного направления токов в катушках.**  
- Если токи текут **в одном направлении** (например, обе по часовой стрелке), катушки **притягиваются**.  
- Если токи текут **в противоположных направлениях**, катушки **отталкиваются**.

Эта двунаправленность позволяет:
- В режиме **притяжения**: сжимать матрицу, повышая локальную жёсткость или вызывая изгиб в сторону сближения.  
- В режиме **отталкивания**: растягивать или «надувать» участок матрицы, создавая выпуклость, расширение или обратный изгиб.

Таким образом, одна и та же EM Muscle может выполнять **два функционально противоположных действия** без механической переконфигурации — достаточно изменить полярность или фазу тока на одной из катушек.

> *Примечание: для независимого управления направлением тока каждая катушка предпочтительно подключается к отдельному каналу драйвера (например, двум полумостам H-моста).*

Эта сила **предварительно напрягает силикон**, повышая сопротивление внешнему давлению — что воспринимается как **уплотнение**.

// Outer dimensions: 200mm wide x 280mm tall x 20mm deep
// Wall thickness: 3mm
// Coil pockets: 15mm diameter x 5mm deep

module face_mold_half(is_left = true) {
    // Main cavity (negative of face)
    difference() {
        // Outer block
        cube([200, 140, 20]);
        
        // Face cavity (simplified oval)
        translate([20, 10, -1])
            scale([1, 1, 1])
            cylinder(h = 22, r1 = 80, r2 = 80, $fn=100);
            
        // Coil pockets (positions match SVG)
        // Eyebrow (left/right)
        if (is_left) {
            translate([60, 40, 15]) cylinder(h=6, r=7.5, $fn=30);
        } else {
            translate([140, 40, 15]) cylinder(h=6, r=7.5, $fn=30);
        }
        
        // Eye (left/right)
        if (is_left) {
            translate([60, 70, 15]) cylinder(h=6, r=7.5, $fn=30);
        } else {
            translate([140, 70, 15]) cylinder(h=6, r=7.5, $fn=30);
        }
        
        // Mouth zone (center)
        translate([100, 110, 15]) cylinder(h=6, r=12, $fn=50);
        
        // Chin
        translate([100, 130, 15]) cylinder(h=6, r=6, $fn=30);
    }
    
    // Alignment pins (2 per half)
    translate([30, 130, 0]) cylinder(h=10, r=2, $fn=20);
    translate([170, 130, 0]) cylinder(h=10, r=2, $fn=20);
}

// Render LEFT half
translate([-210, 0, 0]) face_mold_half(true);

// Render RIGHT half
face_mold_half(false);

### 5. Метод изготовления

1. **Намотать катушки** на пластиковые оправки; герметизировать эпоксидной смолой.  
2. **Расположить в форме** (см. РИС. 4).  
3. **Смешать силикон** (Ecoflex 00-30) + опционально 20% железного порошка рядом с катушками.  
4. **Залить и дегазировать**; выдержать 4+ часа.  
5. **Извлечь из формы** → полностью функциональная EM Muscle.

### 7. Система управления

- **Вход**: Аналоговый сигнал или ШИМ 0–12 В (например, Arduino, Raspberry Pi)  
- **Драйвер**: Массив MOSFET или H-мост L298N с возможностью независимого управления каждой катушкой  
- **Питание**: 12 В @ <500 мА на одну мышцу  
- **Опциональная обратная связь**: Емкостные датчики деформации для замкнутого контура управления жесткостью  

### 8. Вариант с подвижным ферромагнитным сердечником (поршнем)

В альтернативной конфигурации EM Muscle может содержать **подвижный ферромагнитный элемент** — например, цилиндрический **сердечник-поршень (105)**, расположенный coaxially относительно одной или обеих катушек.

- Сердечник выполнен из **мягкого магнитного материала**: чистое железо, пермаллой, композит на основе железного порошка в силиконовой связке.  
- Диаметр: 3–20 мм; длина: 5–40 мм.  
- Сердечник **не жёстко закреплён**, а свободно скользит в канале, сформированном в силиконовой матрице (103).  
- Канал может быть заполнен **вязкоупругой средой** (например, силиконовым маслом) для демпфирования.

**Принцип действия:**  
При активации одной или обеих катушек сердечник испытывает силу, стремящуюся переместить его в область максимальной напряжённости магнитного поля:
- Если активна **одна катушка**, сердечник **притягивается к ней** (режим линейного привода).  
- Если активны **обе катушки с одинаковой полярностью**, сердечник может быть **зафиксирован по центру** (режим упрочнения).  
- При **противофазной активации** сердечник может совершать **осцилляции** или **перемещаться в заданную зону**.

Этот режим особенно полезен для:
- Тактильных интерфейсов (имитация пульса, вибрации)  
- Линейных микродвижений (например, шевеление усами у аниматронного животного)  
- Гибридных приводов, сочетающих **локальное упрочнение** и **выдвижение**

> *Примечание: сердечник может быть частично или полностью покрыт тонким слоем силикона для предотвращения трения и обеспечения биосовместимости.*

## **6. ПРИМЕРНЫЕ ВАРИАНТЫ ОСУЩЕСТВЛЕНИЯ**

### 1. Система мимики андроида  
- 8 EM Muscles под силиконовой кожей толщиной 4 мм  
- Питание от USB (повышение 5 В до 12 В)  
- Выражения: улыбка, хмурость, удивление, отвращение  
- **Режим отталкивания** используется для надувания щёк или поднятия верхней губы; **режим притяжения** — для опускания бровей или уголков рта.

### 2. Протез руки  
- EM Muscles в каждом пальце (антагонистические пары)  
- Управление через ЭМГ-датчик  
- Естественное комплаентное сжатие  
- Возможность **локального расширения** (отталкивание) для адаптации к форме объекта.

### 3. Биомиметическое щупальце осьминога  
- Спиральный массив EM Muscles  
- Перистальтическое волновое движение путём последовательной активации  
- Чередование **притяжения и отталкивания** для создания сложных изгибов и утолщений.

### 4. Тактильный интерфейс с имитацией пульса  
- EM Muscle содержит центральный железный поршень диаметром 5 мм.  
- При импульсной активации катушки поршень совершает короткое выдвижение (0,5–2 мм), имитируя артериальный пульс под кожей.  
- Используется в медицинских симуляторах или эмоциональных роботах для передачи «живости».

### 5. Аниматронное животное с подвижными усами  
- В каждом усе — миниатюрная EM Muscle с сердечником-поршнем.  
- Независимое управление позволяет шевелить усами вперёд/назад, создавая реалистичное поведение.  
- Конструкция остаётся мягкой на ощупь благодаря силиконовой оболочке.

### 6. Гибридный палец с локальным выдвижением  
- Помимо антагонистических EM Muscles для сгибания, в подушечке пальца размещена EM Muscle с поршнем.  
- При захвате хрупкого объекта активируется **режим отталкивания** для адаптивного расширения + **выдвижение поршня** для улучшения тактильного контакта.

## **7. ФОРМУЛА ИЗОБРЕТЕНИЯ**

1. Электромагнитная искусственная мышца, содержащая:  
   a. первую плоскую спиральную электромагнитную катушку;  
   b. вторую плоскую спиральную электромагнитную катушку, параллельную первой и расположенную на расстоянии 0,5–5 см;  
   c. мягкую эластичную полимерную матрицу, полностью инкапсулирующую обе катушки;  
   d. причем подача электрического тока на обе катушки генерирует магнитную силу, создающую предварительное напряжение в матрице, тем самым увеличивая её эффективную жесткость,  
   **причём направление магнитной силы — притяжение или отталкивание — регулируется относительной полярностью токов, подаваемых на первую и вторую катушки.**

2. Мышца по п. 1, где полимерная матрица состоит из силиконового каучука с твердостью по Шору 00-10 до 00-50.

3. Мышца по п. 1, дополнительно содержащая магнитные частицы, распределённые в полимере вблизи катушек.

4. Мышца по п. 1, организованная в антагонистическую пару для обеспечения двунаправленного изгиба или линейного движения,  
   **или выполненная с возможностью переключения между режимами сжатия и расширения за счёт реверсирования полярности тока на одной из катушек.**

5. Гуманоидный робот, содержащий множество мышц по п. 1, встроенных в конечности, лицо или корпус для обеспечения естественных движений и переменной комплаентности.

6. Способ управления жесткостью и формой робота, включающий:  
   - подачу низковольтного сигнала (0–12 В) на катушки по п. 1 для аналоговой модуляции жесткости,  
   - **и выбор режима притяжения или отталкивания путём задания одинаковой или противоположной полярности токов на первой и второй катушках.**

7. Робот по п. 5, где лицо содержит по меньшей мере восемь электромагнитных мышц, сконфигурированных для воспроизведения человеческих мимических единиц, включая лобную, круговую глазную, скуловую, угловую мышцу рта и подбородочную мышцу,  
   **причём отдельные мышцы используют режим отталкивания для создания выпуклостей (например, надувание щёк) и режим притяжения для утягивания тканей (например, опускание уголков рта).**

8. Мышца по п. 1, дополнительно содержащая **подвижный ферромагнитный сердечник**, расположенный coaxially относительно по меньшей мере одной из катушек и выполненный с возможностью линейного перемещения внутри канала, сформированного в полимерной матрице под действием магнитного поля, генерируемого катушками.

9. Мышца по п. 8, где ферромагнитный сердечник выполнен в виде цилиндрического поршня из железа, пермаллоя или композита на основе магнитного порошка, и используется для создания направленного выдвижения, тактильной стимуляции или локального упрочнения.

## **9. ЗАКЛЮЧЕНИЕ**

Электромагнитная искусственная мышца предлагает практичное, безопасное и биомиметическое решение для следующего поколения мягкой робототехники. Она позволяет андроидам и искусственным существам двигаться и ощущаться **более по-человечески** — с переменной жесткостью, бесшумной работой, врожденной комплаентностью и **способностью к двунаправленной активации без механических переключений**.

**Благодаря возможности интеграции подвижных магнитных сердечников, изобретение дополнительно обеспечивает направленное линейное действие — без потери общей мягкости, безопасности и биомиметичности.**

Эта технология преодолевает разрыв между жесткими машинами и живыми тканями.
